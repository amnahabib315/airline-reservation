# Airline Reservation Database Project

A Database Systems project developed as part of Software Engineering coursework.

## Project Contents
- ER Diagram (ERD)
- Table creation scripts (DDL)
- Data insertion scripts (DML)
- SQL queries including joins, subqueries, views and transactions

## Tools Used
- MySQL
- draw.io(for ERD)

## Entities
CITY, AIRPORT, FLIGHT, AIRCRAFT, SEAT, SEAT_CLASS,
PASSENGER, BOOKING, TICKET, CANCELLATION
